The Code


                        //get string from the page
                        //controller function
                        function getValue() {
                        
                            document.getElementById('alert').classList.add('invisible');
                        
                            let userString = document.getElementById('userString').value;
                            
                            let revString = reverseString(userString);
                        
                            displayString(revString);
                        }
                        
                        //reverse the string
                        //logic function
                        function reverseString(userString) {
                        
                            let revString = [];
                        
                            //reverse string using a for loop
                            for (let index = userString.length-1; index >= 0; index--) {        
                                revString += userString[index];
                            }
                        
                            return revString;
                        }
                        
                        //display the reversed string to the user
                        //view function
                        function displayString(revString) {
                        
                            //write to the page
                            document.getElementById('msg').innerHTML = `Your string reversed is: ${revString}`;
                        
                            //show the alert
                            document.getElementById('alert').classList.remove('invisible');
                        }
Rewind Code Explination

The code is structured with a high-order function getValue() being called on when the user clicks on the "Reverse Me" button.

First, getValue() will retrieve the string the user input from The Document Object Model (DOM) and assign it to the variable "userString".

Next, the function reverseString() will be called on with "userString" as its argument.

reverseString() will use a for loop to create a new string called "revString" and assign it the letters from userString starting from the last letter and ending with the first.

revString() will then return "revString" to getValue() and be assigned as "revString", a variable in the scope of getValue().

Last, the displayNumbers() function is called with "revString" as its argument.

displayNumbers() will add "revString" into the DOM inside the div named "msg", and remove the class "invisible" from the div named "alert" to display the string reversed.
